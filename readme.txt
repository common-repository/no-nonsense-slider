A simple lite weight plugin for use with the theme from SwiftThemes.Com

== Description ==

Install the plugin and configure it from the customiser. This plugin doesn't have a dedicated options page.



== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/eaa` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. This plugin has no admin or settings page. It can be configured from the customiser when you are using the PageSpeed theme.


== Frequently Asked Questions ==

= Is it only for themes released by SwiftThemes.Com =
Yes, though it works with other themes it is developed for use with theme built by SwiftThemes.Com
